# **App Name**: RentEase

## Core Features:

- Car Listing: Display a list of available cars with details like model, year, and price.
- Booking Calendar: Enable users to select rental dates using a calendar interface.
- Rental Booking: Process rental bookings, capturing user details and rental dates.
- Car Recommendation Tool: Suggest cars that are a good fit, by letting the user describe their preferences (e.g. fuel efficiency, seating capacity) in their own words.

## Style Guidelines:

- Primary color: Deep sky blue (#00BFFF) for a trustworthy and open feel.
- Background color: Light gray (#F0F8FF), a very light tint of the primary color, almost white, but not quite.
- Accent color: Royal blue (#4169E1), a more saturated analogous color, is chosen for UI controls like buttons and active elements.
- Headline font: 'Poppins' sans-serif for headlines, creating a geometric and fashionable feel.
- Body font: 'PT Sans' sans-serif is used for body text, to create a readable, modern feel.
- Use car-related icons (steering wheel, map pin, etc.) for easy navigation.
- Subtle transition effects when navigating between car listings.
